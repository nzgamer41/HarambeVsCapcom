Instalation:
Make sure you have the latest version of M.U.G.E.N.
If not, check out www.elecbyte.com

After installing M.U.G.E.N, create a directory under the "CHARS" directory.
The directory must be called "zangief"
Unzip the files in this archive there.

Edit SELECT.DEF under the DATA directory as instructed within the file,
adding worm with the line:

zang, stages/_____.def

..where _____ is which background file you choose.
You can also write this like that:

Zang, random 

Random means here the MUGEN will randomly chose a stage.

Movelist for Zangief:

Legend:

(air)		- Move can be done also in air
(guard)		- Move can be done only while guarding
(air only)	- u don't understand this?

f - forward
b - back
u - up
d - down

P - Any Punch
K - Any Kick

LP 	- Low  Punch
MP 	- Mid    " 
HP 	- High   "

LK 	- Low  Kick
MK 	- Mid   "
HK 	- High  "

PP 	- Two Punches
KK 	-  "   Kicks

PPP 	- Three Punches 
KKK 	-   "   Kicks

qcf 	- quarter circle forward - d,df,f
qcb 	-    "      "    back    - d,db,b

hcf 	- half circle forward 	- b,db,d,df,f
hcb 	-  "     "    back	- f,df,d,db,b

360	- full circle (any directory)	- for example: b, db, d, df, f, uf, u
